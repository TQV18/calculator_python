# Calculator App

Một ứng dụng máy tính đơn giản được viết bằng Python và thư viện CustomTkinter.

## Chức năng
- Thực hiện các phép toán cơ bản: +, -, *, /
- Hỗ trợ số thập phân
- Hỗ trợ phần trăm (%)
- Giao diện tối (dark mode)

## Công nghệ sử dụng
- Python
- CustomTkinter
